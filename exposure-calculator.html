曝險計算機

Exposure Rate Calculator — 現股・槓桿ETF・融資・質押

曝險率是衡量你的股票部位相對於自有淨資產的比率，代表市場波動對你整體財富的實際放大倍數。 超過 100% 代表你已使用槓桿——市場的漲跌對你的影響，都會被等比例放大。

曝險率 = 總股票曝險 ÷ 自有淨資產 × 100%

總股票曝險 = 現股市值 + 槓桿ETF有效曝險 + 融資總市值 + 質押股市值

總資產 = 現股市值 + 槓桿ETF市值 + 融資總市值 + 質押股市值 + 現金

自有淨資產 = 總資產 − 融資借款 − 質押借款 − 信貸借款

槓桿ETF有效曝險 = 市值 × 槓桿倍數（如 00631L 持有100萬 → 曝險200萬）

融資自備款 = 融資總市值 − 融資借款　｜　質押股淨值 = 質押股市值 − 質押借款

曝險率 狀態 說明
< 50% 保守 大量現金緩衝，市場波動影響有限
50–100% 穩健 以現股為主，仍保有部分現金
100–150% 積極 輕度槓桿，下跌損失超過本金比例
150–200% 高曝險 中度槓桿，需嚴格控管部位
> 200% 危險 重度槓桿，市場微幅震盪即可造成重大損失
例：曝險率 200%，市場下跌 10% → 你的淨資產損失 20%。
若持有 00631L（2倍槓桿ETF）100萬，其有效市場曝險為 200萬，計入曝險率計算。

融資 向券商借款買股，自備款 + 借款 = 總市值
融資借款不得大於融資總市值。
✓ 填入：用融資（向券商借款）買入的股票部位，填「總市值」與「融資借款金額」，工具會自動算出你的自備款
✗ 不要填：融資買的股票不要同時再填進現股欄，否則曝險會計算兩次
質押 以持股作擔保向銀行/券商借款
質押借款不得大於質押股市值。
✓ 填入：拿去質押的股票（目前市值）與質押借款金額；質押股你仍然持有，漲跌你全部承擔
✗ 不要填：已填在質押欄的股票不要再填一次現股欄，否則市值與曝險都會算兩次
信貸 向銀行借的個人信用貸款
✓ 填入：向銀行申請的個人信用貸款餘額（尚未還清的部分）
若信貸資金已買股 → 股票填在現股欄，信貸借款填這裡；若還放在帳戶 → 現金也照填，信貸借款仍填這裡
✗ 不是這裡：房貸、車貸等有實體抵押品的貸款通常不影響投資曝險，可不填
現金 投資帳戶或銀行可動用現金
✓ 填入：券商交割帳戶餘額、銀行可動用現金，這是你的緩衝，越多則曝險率越低
✗ 不要填：已借出去的質押借款或融資金額不算現金（那是負債，已在各欄借款中扣除）
月收支 用於計算收入還債能力與流動比率
月收入：每月稅後實拿薪資，用於「還清負債需幾個月薪水」
月付款：每月固定需支付的借款（融資利息、質押利息、信貸月繳等）合計，用於計算流動比率（現金能撐幾個月）與月現金流
